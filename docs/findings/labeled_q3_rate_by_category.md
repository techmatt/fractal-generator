## The confound — do not read the pooled table as a per-family price list

These batches were **not sampled uniformly**, so the pooled q3 rate is not "how easy is it to find a good one in family X."

- **mandelbrot** is the only family with a quasi-unbiased draw: `2026-06-23_flat_generate_loose0_v3` (loose accept gate over flat redraws, no model/descent selection). Its q3 rate is the least-biased mandelbrot base rate available.

- **julia / multibrot / phoenix STILL have no *unbiased* labeled draw**, but they are no longer unlabeled here — the previous version of this report read in-row `label.score` + batch `scores.json` only and so dropped every sidecar-only batch (`julia_ladder_j0`, `mining`, `scale`, `jm3_band`, `jm45_band`). Those labels exist and are counted now. What remains true is that every labeled draw in these families is *biased*, not that they are unlabeled.

- **`julia_ladder_j0` (1000 J0 Julia locations) is NOT an unbiased base rate**, despite its size. Regime (`tools/julia_ladder/build_j0.py`): its Julia `c` values are the **label-2/3 (good) Mandelbrot centers** from the v4 manifest (893 raw → 618 deduped neighborhoods); each seed then spawns systematic center-zoom rungs (`fw = 3.0/4`, `3.0/8`) **and** `guided-descend` descent rungs, v4-scored, then stratified-sampled to 1000. So it is a systematic zoom ladder *conditioned on already-good `c`* with a descent-selected half — positively biased toward good Julia. Its q3 rate is an over-estimate of the Julia base rate, not the base rate. Julia's true unbiased rate was **not** available all along.

- The only labeled multi-family harvest, `2026-07-05_gather_v6`, is a guard-OFF harvest oversampled by raw v5 rank — positively biased. `jm3_band` / `jm45_band` are model-band revival batches (sampled inside a targeted score band), also biased. So the honest *unbiased* per-family rate for julia / multibrot / phoenix is still **unknown** — but the labels themselves are present and pooled above.

- `2026-07-12_blindspot_v6reject_v1` is a **negative set by construction** (v6-rejects); its near-zero q3 rate is by design and drags the pooled mandelbrot rate down.

- `guided_descend_rev4`, `rev4occfix_v2filtered`, and `mining_v3guided_v1` are descent/model-selection-biased (positively enriched).
